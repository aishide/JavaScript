# JavaScript

## Description
This repository contains small web projects created using **HTML, CSS, and JavaScript**.  
The projects demonstrate how JavaScript is used to add **logic, interactivity, and dynamic behavior** to web pages.

Each project focuses on implementing JavaScript concepts in a simple and practical way.

- JavaScript

- Event handling

3. Interact with the page to see JavaScript in action.
